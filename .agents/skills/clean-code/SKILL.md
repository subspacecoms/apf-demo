# Clean Code Workflow

## Steps

### 1. Inspect Variable Names
Inspect variable names in the context and suggest improvements for readability.

### 2. Inspect Method Parameters
Inspect the number of parameters of each method. If a method has more than 5 parameters, suggest a refactoring plan.

### 3. Inspect Method Length
Inspect the number of lines in each method. If a method is longer than 150 lines, suggest a refactoring plan.

### 4. Inspect File Length
Inspect the number of lines in each single file. If a file is longer than 500 lines, suggest a refactoring plan.

### 5. Inspect SOLID Principles
Inspect the code for compliance with SOLID principles and suggest updates if improvements can be made.

### 6. Inspect Magic Strings
Inspect the code for magic strings and suggest moving them to configurations.
